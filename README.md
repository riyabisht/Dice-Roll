# Dice-Roll

## About

This is a basic Android Application using Kotlin. 

## how to run 

- download ZIP and unpack it 
- Start `Android Studio`
- select `empty` template
- Go to `files` --> `new` -->`import Project`
- Navigate to your folder
- `Run`
